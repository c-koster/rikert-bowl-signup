Rikert Bowl Signups

Hello! This is a simple web app to help with rikert and snowbowl signups. A link to the Github repository can be found here, and some code examples on getting it to work are shown below.

Input Parameters

first - your fisrt name
last - your last name
email - your midd email address
id - the pass number on your id -- put 00000 if not applicable
day - the date you're looking to sign up for in (YYYY-MM-DD). Defaults to today's date. this isn't working yet btw
isBowl - put 1 if you want to sign up for the snowbowl, 0 for rikert

Here is a python example:

import requests
import json
data = {'first': 'culton','last': 'koster','email':'ckoster@middlebury.edu','id':'00000','isBowl':0}
r = requests.post('https://rikert-bowl-signup.herokuapp.com/',json=json.dumps(data))
print(r.text)
